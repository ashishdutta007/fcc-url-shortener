URL Shortener Microservice API

What is this?

This is a microservice API project for Free Code Camp that will accept a URL and return a shortened URL response.If input URL is invalid or duplicate response returns an error message.

Usage:

https://frozen-reaches-95347.herokuapp.com/new/http://www.google.com/

Example Response:

					{
						"originalURL":"http://www.twitter.com/",
						"urlCode":1002,
						"shortURL":"pE.theo"
					}
